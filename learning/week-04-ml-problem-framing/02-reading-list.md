# Danh mục đọc Tuần 4

## Bắt buộc

1. [Google ML — Introduction to Machine Learning Problem Framing](https://developers.google.com/machine-learning/problem-framing)
   - Mục tiêu đọc: nhận biết ML có phù hợp và chuyển bài toán thành input/output/metric.
   - Câu hỏi: bài toán của bạn tạo decision nào?

2. [Google ML — Understand the problem](https://developers.google.com/machine-learning/problem-framing/problem)
   - Mục tiêu đọc: bắt đầu từ mục tiêu phi-ML, actionability, dữ liệu tại prediction time và heuristic benchmark.
   - Bài ghi chú: viết một phương án non-ML cho Project 1.

3. [Google ML — Frame the ML problem](https://developers.google.com/machine-learning/problem-framing/ml-framing)
   - Mục tiêu đọc: ideal outcome, model goal, output và success metrics; nhận diện giới hạn proxy label.
   - Bài ghi chú: khóa target/window và output type.

4. [Google — Rules of Machine Learning](https://developers.google.com/machine-learning/guides/rules-of-ml)
   - Đọc Rule #1–#10.
   - Bài ghi chú: liệt kê metric/baseline cần dựng trước model phức tạp.

5. [scikit-learn — DummyClassifier](https://scikit-learn.org/stable/modules/generated/sklearn.dummy.DummyClassifier.html) và [DummyRegressor](https://scikit-learn.org/stable/modules/generated/sklearn.dummy.DummyRegressor.html)
   - Mục tiêu đọc: hiểu most-frequent, stratified, mean và median baseline.
   - Câu hỏi: dummy baseline nào phù hợp với target của bạn?

## Khuyến nghị

6. [Google — Managing ML Projects](https://developers.google.com/machine-learning/managing-ml-projects)
   - Tập trung vào các pha ideation, planning, experimentation và productionization.

7. [TensorFlow — Responsible AI](https://www.tensorflow.org/responsible_ai)
   - Xem evaluation theo slices, data/model documentation và monitoring.

8. Aurélien Géron, *Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow*, Chương 1.
   - Ôn supervised/unsupervised, batch/online, instance/model-based learning và các thách thức chính.
