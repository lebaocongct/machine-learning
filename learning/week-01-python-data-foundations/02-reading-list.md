# Reading list Tuần 1

## Cách đọc

Không cần đọc toàn bộ tài liệu. Dùng câu hỏi định hướng và dừng khi đã tự giải thích, viết lại ví dụ và áp dụng vào lab.

## Bắt buộc — khoảng 90 phút

### NumPy — 30 phút

1. [NumPy: the absolute basics for beginners](https://numpy.org/doc/stable/user/absolute_beginners.html)
   - Đọc: array creation, indexing, shape/axis, basic operations.
   - Tự trả lời: `shape`, `ndim`, `size`, `dtype` khác nhau thế nào?
2. [Broadcasting](https://numpy.org/doc/stable/user/basics.broadcasting.html)
   - Đọc: general broadcasting rules.
   - Viết ra giấy shape của ba phép toán thành công và hai phép toán thất bại.

### Pandas — 35 phút

1. [10 minutes to pandas](https://pandas.pydata.org/docs/user_guide/10min.html)
   - Tập trung: object creation, viewing data, selection, missing data, operations, merge, grouping.
2. [Working with missing data](https://pandas.pydata.org/docs/user_guide/missing_data.html)
   - Tập trung: cách Pandas biểu diễn missing theo dtype.
3. [Merge, join, concatenate and compare](https://pandas.pydata.org/docs/user_guide/merging.html)
   - Tập trung: database-style joins và validation/cardinality.

### Visualization và test — 25 phút

1. [Matplotlib quick start](https://matplotlib.org/stable/users/explain/quick_start.html)
   - Tập trung: `Figure`, `Axes`, labels và save figure.
2. [pytest — Get Started](https://docs.pytest.org/en/stable/getting-started.html)
   - Tập trung: test discovery, `assert`, exception và `pytest.approx`.

## Thiết lập — khoảng 20 phút

1. [Python `venv`](https://docs.python.org/3/library/venv.html)
   - Hiểu vì sao virtual environment có thể xóa và tạo lại.
2. [Installing Jupyter](https://jupyter.org/install)
   - Chỉ cần phần JupyterLab.
3. [Pro Git — Getting a Git Repository](https://git-scm.com/book/en/v2/Git-Basics-Getting-a-Git-Repository)
   - Tập trung: `git init`, `status`, `add`, `commit`.
